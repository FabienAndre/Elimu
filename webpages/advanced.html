Advanced

To help spice up your Karel programs we have granted Karel some bonus functionality. We have given Karel the ability to paint and to perform actions randomly.

The new command paintCorner('color') will floodfill Karel's current corner. Color must be an HTML color code. For information on HTML color codes click here. Note that the color must be surrounded by quotes.

The new command randomChance(p) is a new condition statement that will evaluate to true with probability p. If no value of p is provided it assumes the default value of 0.5.

Here's an example that uses both the advanced Karel features:


7x7.w,470,303,advanced
