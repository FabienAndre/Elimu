While Loops

The for statement is useful only when you know in advance the number of repetitions you need to perform. In most applications, the number of repetitions is controlled by the specific nature of the problem. It would be much better if Karel could continue to execute a block of code until it encountered some condition that caused it to stop, such as reaching the end of the world.

To accomplish this task, you need to use a while statement. In Karel, a while statement has the following general form:

    while(test) {
        statements to be repeated
    }

Where test is one of the questions, listed in the conditions page, Karel can ask about its current environment. If the test passes when Karel gets to the while loop, the code in the body is executed. Every time the code in the body finishes executing, the test is rechecked and if it still holds true the computer executes the code in the body again.
